# Timișoara App -- MVP Feature Breakdown

## Phase 1: MVP (Minimum Viable Product)

### 1. Interactive Map with Points of Interest
- Display Timișoara on an interactive map (OpenStreetMap tiles)
- Show categorized POIs: landmarks, parks, museums, churches, restaurants, cafes
- Tap a POI to see details: name, description, photos, address, hours, rating
- Search and filter by category
- GPS location support ("near me")

### 2. Events Calendar
- List upcoming events in Timișoara (concerts, theater, exhibitions, festivals)
- Filter by category: music, theater, art, sports, food, family-friendly, free
- Event detail page: date/time, venue, description, ticket link
- Bookmark/save events
- Calendar and list view toggle

### 3. Public Transport (STPT)
- Browse all STPT lines (tram, bus, trolleybus)
- View schedules for each line and direction
- See route on map with stops
- Basic route planning: "How do I get from A to B?"
- Ticket prices and purchasing info

### 4. Restaurant and Cafe Directory
- List restaurants and cafes with categories (Romanian, international, vegan, etc.)
- "Open now" filter based on operating hours
- Sort by distance, rating
- Detail page: menu highlights, photos, address, phone, hours
- Cuisine-type filters

### 5. User Accounts
- Sign up / sign in (email + social login)
- Profile: local vs. tourist mode
- Save favorite places and events
- Language preference (Romanian, English)

---

## Phase 2: Community & Civic

### 6. Civic Reports
- Report a problem (photo + GPS + category)
- Track report status
- View nearby reported issues on map

### 7. News & Weather
- Local news feed aggregation
- Weather widget (7-day forecast)
- Push notifications for alerts

### 8. Neighborhood Guides
- Detailed guides for each neighborhood (Cetate, Fabric, Iosefin, Elisabetin)
- History, character, key places
- Walking tour routes

---

## Phase 3: Social & Advanced

### 9. Community Forums
- Neighborhood discussion boards
- Local marketplace (buy/sell)
- Volunteer opportunities

### 10. Accommodation
- Hotel/hostel listings aggregation
- Day trip suggestions

### 11. Health & Emergency
- Hospital/pharmacy finder with hours
- Emergency numbers directory

### 12. Gamification & Advanced Features
- Visited places tracker with badges
- AR landmark overlay (stretch)
- Offline maps
- Additional languages (German, Hungarian, Serbian)
